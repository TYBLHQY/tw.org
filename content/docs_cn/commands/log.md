# log

`log` 命令与 [`add`](../add/) 命令相同，但有一个区别 - 任务弅以完成状态添加。
有时候有必要记录已经完成的任务，如果您正幸追跑学。
例如：

```
$ task log 订购材料 +home +construction
```

此任务立即被记录为完成，这意味着它不会出现在任何显示待处理任务的报告中。

```
$ task log 订购材料 start:2017-02-05 end:2017-02-06
```

此任务立即被记录为完成，有一个开始日期和一个结束日期。

## 另请参阅

其他创建任务的方式包括：

- [`add`](../add/) 命令
- `duplicate` 命令
- `import` 命令
